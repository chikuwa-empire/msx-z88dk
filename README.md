# msx-z88dk
<p>z88dkによるクロス開発環境でMSX0のアプリを作って楽しむリポジトリーです。</p>

## ライブラリ
### msx0.h msx0.c
MSX0 StackのIOT命令をC言語から使うための関数群です。
### msx0lcd.h msx0lcd.c
キャラクタLCD(1602)をMSX0から使うための関数群です。

## サンプル
### envsampl.c
温湿度センサー(SHT30)のサンプルプログラムです。
### lcdsampl
キャラクタLCD(1602)のサンプルプログラムです。
